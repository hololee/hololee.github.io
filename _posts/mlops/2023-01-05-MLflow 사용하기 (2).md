# MLflow Tracking Server (UI)











## 세팅하면서 마주했던 오류들

### psql 연결문제

mlflow tracking server에 가능한 [시나리오 5번](https://mlflow.org/docs/latest/tracking.html)에서 psql 접속시 SQLAlchemy 포멧을 따라주어야 한다.

```
postgresql://아이디:비밀번호@서버주소/DB이름
```



### ui 런

mlflow server \
  --backend-store-uri postgresql: \
 --artifacts-destination s3://recordings-agora-ai/mlflow \
  --host 0.0.0.0
